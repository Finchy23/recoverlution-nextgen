# Figma UI Return Checklist

Use this as the return shape for the next Figma handoff.

## Companion
- final shell refinement
- final `READ` room
- final `ECHO > PLOT` room
- final `ECHO > MAP` room
- final `ECHO > LINK` room
- notes on copy, motion, or hierarchy that code must preserve

## Professional Console
- top-level IA
- home/workbench view
- one schedule/sessions view
- one practitioner/client detail flow

## Organisational Core
- top-level IA
- home/operations view

- one org settings/policy view

## Guardrail check
- Companion still feels intimate
- Console still feels operational
- Core still feels administrative
- no library creep in `READ`
- no settings-page creep in `ECHO`
- no admin gravity in Companion
