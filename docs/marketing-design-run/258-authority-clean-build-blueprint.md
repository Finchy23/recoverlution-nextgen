# 258 Authority Clean-Build Blueprint

- why lecture-driven wellness fails under real friction
- how rigor and restraint coexist in this system

2. Structured Argument Block
3. Evidence Lens
4. Architecture Reveal (supporting, not dominant)
5. Gentle Conversion Block
