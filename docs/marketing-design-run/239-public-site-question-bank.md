## Purpose
- give the design, asset, copy, and frontend lanes one shared set of questions for the public-site clean-build run
- stop page work from collapsing into local taste or old-homepage imitation

## The questions visionary teams must answer

### Brand / category
1. What does the page make inevitable before it makes explicit?
2. What is the smallest category truth we must establish on first contact?
3. What is the one feeling the site should create before it asks for trust?
4. What should a visitor repeat in their own words after one scroll?

### Hero
5. What burden belongs to the hero asset, and what burden belongs to typography?
6. If the asset disappears, what still holds?
7. If the copy gets cut in half, what still holds?
8. What does the hero refuse to explain?
9. What must the hero never feel like: ad, trailer, app demo, or wallpaper?

### Motion
10. What motion tells the truth of the system?
11. What motion is only performance theatre?
12. What should happen in reduced-motion mode without meaning loss?
13. Where should the glass feel alive, and where should it become quiet?

### Trust
14. What is the first moment the site proves it understands real weight?
15. What is the first moment the site proves it is not another wellness shell?
16. How do we make proof visible without becoming bureaucratic?
17. What should be deferred out of the homepage and into Trust Chain pages?

### Information architecture
18. Which pages create desire?
19. Which pages create belief?
20. Which pages create evidence?
21. Which pages close objections?
22. Which pages should never carry the burden of category creation?

### Asset system
23. Which asset family owns the entrance?
24. Which family explains continuity?
25. Which family explains weight, structure, and transformation?
26. Which family should never be used in the same section as another?
27. What asset should be treated as a system signal, not a hero?

### Copy
28. Which public words are system words, not campaign words?
29. Which words should never appear on first-screen public surfaces?
30. What is the longest a section can speak before it breaks the calm?
31. What lines belong to public doctrine and what lines belong only to internal doctrine?

### Commercial path
32. What must be true before a visitor reaches `/pricing`?
33. What must be true before a visitor reaches `/enter`?
34. How does the site create desire without sounding like a funnel?
35. What should the commercial path never promise while auth and hero/media remain conditional?

### The hard questions
36. What if the strongest thing on the page is silence?
37. What if the page is better after removing a section?
38. What if the current homepage taught us the wrong lesson?
39. What if the most â€śbeautifulâ€ť option is less trustworthy?
40. What if coherence matters more than surprise?

## Decision rule
- no page decision is accepted until it answers the relevant questions for its family
- unresolved answers must be logged, not guessed
