# 249 Public Module-to-Route Map

Purpose: map the shared public-site module vocabulary to the actual route spine so design and frontend can build from a stable system instead of improvising page structure.

## Launch Spine

### `/`
Allowed primary modules:
- Category Lockup
- Bridge Pause
- Family Window

- Architecture Reveal
