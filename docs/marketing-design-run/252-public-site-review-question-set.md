# 252 Public Site Review Question Set

Purpose: define the review questions that should be asked of every public surface before it is treated as stronger than the current live route.

## Questions
1. What is this page trying to make a person feel in the first five seconds?
2. What single burden is this page carrying?
3. What would become confusing if this page were removed from the route spine?
4. What is the dominant module on this page, and is it carrying the right burden?
5. Is the page trying to prove too much too early?
6. Is the page using assets or motion to compensate for weak structure?
7. Does the copy feel consumer-first or expert-first?
8. What is the next action, and is it calm and honest?
9. What unresolved doctrine does this page currently depend on?
10. Does this page feel like the same company as the homepage without repeating the homepage?

## Failure answers
A page is not ready if the answers reveal:
- multiple competing burdens
- weak next action
- unsupported proof claims
- homepage repetition instead of family behavior
- unresolved doctrine hardened into surface truth
